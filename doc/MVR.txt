MVR - mobile video recorder
Предназначен для записи видеоматериала с камер (как IP таки и встроеннной) на носитель.

Функциональные возможности в порядке разработки.

1. Запись на носитель видео потоков H264 с произвольного количества IP камер c нарезкой на видеофайлы заданной длительности (аудио?).

2. Передача видеофайлов на сервер (SFTP or SSH?)
 - автоматическое подключение к известным сетям (WiFi, 3G)
 - установление связи с сервером
 - выгрузка файлов (FTP?)

3. Отображение на экране живого видео с не менее чем с 4 камер
 - управление layout'ами
 - быстрое переключение layout'ов

4. Отображение, кодирование и запись видео с встроенной камеры (V4L?).

5. Информационное сопровождение маршрута (Информатор)
 - установление связи с сервером
 - получение с сервера данных о маршруте (звуковые и световые сообщения, навигация для водителя?)
 - выдача сообщений по командам водителя и данным геолокации (эмулятор табло на PC, эмулятор GPS приёмника в устройстве)

6. Обеспечение трансляции видеоматериалов через беспроводную сеть (протокол?)
 - обеспечение маршрутизации или прокси сервер

7. Запись данных геолокации, привязка видеоматериалов к данным геолокации, выгрузка данных геолокации на сервер

8. Просмотр записанного видео без выгрузки на сервер
 - просмотр через беспроводную сеть
 - просмотр на устройстве

9. Обеспечение аутентификации водителя, защиты данных (видео, геолокация)

10. Запись "тахографических"  данных

11. Диагностика оборудования.
